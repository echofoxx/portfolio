Integration principles

  1. Every integrated field has one authoritative owner.
  2. Canonical UUIDs and human-readable identifiers are retained.
  3. Adapters must implement retries, idempotency, reconciliation, error handling, lineage, and audit.
  4. ProjectOS, ServiceNow, and this application may not independently overwrite the same authoritative field.
  5. Business events and synchronization contracts are versioned before production use.
